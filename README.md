# txgh-js-bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that When a push event occurs, upload the source file to Transifex and commit the translation file to GitHub.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
